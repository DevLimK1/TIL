# <문자열>

- ### 따옴표 3개(""")로 여러줄 만들기

```python
print("""
안녕하세요
안녕하세요
안녕하세요
""")
```

    안녕하세요
    안녕하세요
    안녕하세요

```python
print("""\
안녕하세요
안녕하세요
안녕하세요\
""")
```

    안녕하세요
    안녕하세요
    안녕하세요
